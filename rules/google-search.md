Prefer `google_search` over built-in web search for research, current events, documentation, and anything where citations add value.

Falls back to built-in web search for quick lookups (or increased accuracy) or if google_search is unavailable.

Do NOT use Google dork operators (site:, filetype:, inurl:, intitle:, intext:, ext:, cache:) — Prefer natural language. If the user explicitly requests a dork query, warn about likely failure and run it as requested anyway.
